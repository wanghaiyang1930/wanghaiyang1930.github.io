# 1. Heuristic Learning

启发式学习核心思想： 通过经验法则、试错和近似策略来解决问题或做出决策，而非依赖穷举搜索或精确算法。

主要特点：
- **效率优先** — 牺牲一定的最优性，换取在合理时间内找到"足够好"的解
- **经验驱动**— 利用领域知识、历史模式或直觉来缩小搜索空间
- **适应性强** — 适用于问题空间过大、无法精确求解的场景


常见应用方向：
- **搜索与优化** — 如 A* 算法中的启发函数、模拟退火、遗传算法
- **强化学习中的探索策略** — 用启发式引导 agent 的探索方向
- **元学习 / 学会学习** — 学习如何生成更好的启发式规则本身
- *游戏 AI** — 棋类博弈中的评估函数设计


与机器学习的交叉：

近年来 "learned heuristics" 方向比较活跃——用神经网络来学习启发式函数（比如用 GNN 学习组合优化问题的分支策略），替代人工设计的规则。
